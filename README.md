# Perception Simulator landing

Static landing page for the Perception Simulator Chrome extension.

## Files

- `index.html` — landing page and interactive extension UI demo.
- `privacy.html` — public privacy policy suitable for the Chrome Web Store privacy URL.
- `styles.css` — responsive design styles.
- `site.js` — interactive popup demo.
- `.github/workflows/pages.yml` — automatic GitHub Pages deployment.
- `DEPLOY.md` — step-by-step deployment instructions in Russian.

No build step or external dependency is required. The install buttons intentionally point to the landing-page root (`./`) until a Chrome Web Store URL is available.
